# CG Translations — Portfolio 🎬

Landing page premium de perfil profesional y catálogo de servicios para Camila Terminiello, Traductora Pública e Intérprete especializada en traducción audiovisual y técnico-científica (Inglés -> Español).

El diseño está inspirado en una estética editorial cinematográfica pesada ("Flyer Style"), utilizando tipografías brutas, bloques de lectura contrastantes, animaciones interactivas fluidas y un esquema de color de alto impacto basado en rojo intenso, amarillo marquesina de Broadway y negro cine.

## Características y Secciones

* Hero Moderno: Impacto tipográfico escalado con efectos dinámicos de resplandor neón en texto calado (text-stroke).
* Broadway Quote: Componente interactivo que simula una marquesina de cine clásica con luces perimetrales intermitentes mediante CSS puro.
* Grilla Editorial (About & Why Us): Disposición limpia en dos columnas para la propuesta de valor e identidad de marca.
* Services Grid: Tarjetas con inversión de contenido al hacer hover (el título oculta su posición para revelar una descripción técnica fluida) e íconos interactivos con efecto elástico (bounce).
* Industries Serve List: Lista de sectores objetivo estructurada como créditos finales cinematográficos, usando íconos dinámicos parametrizados mediante lucide-react.
* Contact Zone: Bloque de links directos optimizados con badges estructurales de zona horaria (GMT-3).
* Responsive Design: Maquetación adaptada al 100% para dispositivos móviles y tablets mediante Media Queries fluidas.

## Stack Tecnológico

* React 19
* Vite (HMR integrado para desarrollo ultra rápido)
* CSS3 Puro (Custom properties globales, Flexbox, CSS Grid, keyframes avanzadas y curvas cubic-bezier para animaciones físicas orgánicas)
* Lucide React (Paquete de vectores de línea consistente para la interfaz)

## Instalación y Desarrollo Local

1. Clonar el repositorio:
git clone [https://github.com/tu-usuario/tu-repositorio.git](https://github.com/tu-usuario/tu-repositorio.git)
2. Instalar las dependencias del proyecto:
npm install
3. Levantar el servidor de desarrollo local con Vite:
npm run dev
4. Compilar y optimizar para producción (Genera la carpeta dist):
npm run build

## Despliegue (Deploy)

Este proyecto está configurado para deployarse de forma nativa en Vercel mediante integración continua con GitHub. Cada push a la rama main compila y actualiza automáticamente el entorno de producción.
